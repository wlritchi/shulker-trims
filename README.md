# Shulker Trims

Trims aren't just for your armor anymore! Add flair to your shulker boxes so you can finally tell them apart. It's a new use for the same old trims!

![A collection of shulker boxes with various colors and trims in a room](https://raw.githubusercontent.com/wlritchi/shulker-trims/main/docs/images/gallery.png)

### Compatibility

Clients with this mod can join vanilla servers, and servers with this mod can host vanilla clients. One jar supports Fabric clients and both Fabric and Paper servers.
