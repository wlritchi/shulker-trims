# Shulker Trims

Trims aren't just for your armor anymore! Add flair to your shulker boxes so you can finally tell them apart. It's a new use for the same old trims!

![A collection of shulker boxes with various colors and trims in a room](https://raw.githubusercontent.com/wlritchi/shulker-trims/main/docs/images/gallery.png)

The smithing table now accepts shulker boxes in the gear slot. All 18 vanilla trims and all 10 trim materials are supported, and trimming a shulker box preserves its contents. You can also re-dye trimmed shulker boxes without affecting the trim.

![A Wild armor trim, blue shulker box, and redstone dust in the smithing table GUI, with a trimmed shulker box in the output slot](https://raw.githubusercontent.com/wlritchi/shulker-trims/main/docs/images/howto.png)


### Compatibility

Clients with this mod can join vanilla servers, and servers with this mod can host vanilla clients. One jar supports Fabric clients and both Fabric and Paper servers.
